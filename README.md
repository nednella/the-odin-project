# The Odin Project

The projects I built while working through [The Odin Project](https://www.theodinproject.com/).

I've gathered them here into a single repo, keeping the full commit history for each. They're listed below in the order I worked through them.

### Foundations

- [Recipes](./recipes) — a small multi-page recipe site
- [Landing Page](./landing-page) — a page rebuilt from a design spec
- [Rock Paper Scissors](./rock-paper-scissors) — the game, against the computer
- [Etch-a-Sketch](./etchasketch) — a browser drawing pad
- [Calculator](./calculator) — a working on-screen calculator

### Intermediate HTML and CSS

- [Sign-up Form](./signup-page) — a styled form with validation
- [Admin Dashboard](./admin-dashboard) — a dashboard layout in CSS Grid

### JavaScript

- [Library](./library) — keep track of books you've read
- [Tic Tac Toe](./tic-tac-toe) — two-player game
- [Restaurant Page](./restaurant-page) — a page rendered entirely in JavaScript
- [To-Do List](./todo-list) — task manager with localStorage
- [Weather App](./weather-app) — live forecasts from a weather API
